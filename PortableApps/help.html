ConEmu Portable Help

windows console emulator / local terminal

ConEmu is a console window (terminal emulator) and tabbed environment for Windows. ConEmu presents multiple consoles and simple GUI applications as one customizable GUI window with tabs, splits, ANSI colors, a status bar and other features. Also, ConEmu can "replace" default Windows terminal, so all console applications started from Explorer (for example) will be seamless opened in ConEmu tab. Learn more about ConEmu...

Please note, that this version of ConEmu Portable is not intended for Far Manager users because this pack does not contains ConEmu's plugins for Far Manager. If you need them, please install full ConEmu version.

Getting ConEmu up

When you run ConEmu first time it will start some default shell, cmd.exe in most cases. Right-click on title bar and choose "Settings..." item, select "Tasks" in the page tree and set up your favorite shells (powershell, git-bash, Visual Studio build environment, and so on). Note, each task may contain more than one shell! Now select "Startup" in the page tree, choose option "Specified named task" and select your favorite from drop-down list.

Don't forget to press "Save settings" button.

Well, your shell will be started next time you run ConEmu.

You may run any named task or even any arbitrary command by pressing [+] button on tab-bar or status-bar.

ConEmu has internal autoupdate feature, you may enable it in "Settings" dialog.
